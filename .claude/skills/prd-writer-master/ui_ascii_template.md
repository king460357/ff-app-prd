# UI ASCII 线框图模板

> 用 ASCII 字符画出页面骨架，标注各区域名称和作用。
> 适合放进飞书/Lark 文档（用代码块包裹），保持等宽字体即可正常显示。

## 绘制要点

### ✅ 推荐做法

1. **标注每个区域的名称和作用**：不只画框，还要写"这是什么"
2. **关键按钮用方括号标识**：`[确认]`、`[取消]`
3. **单选/多选用 ☐ ○ 等符号**
4. **下拉框用 `▼` 标识**
5. **必填项用 `*` 标识**：`名称 *`
6. **保持对齐**：用空格补齐，让框线对齐

### ❌ 避免做法

1. ❌ 只画框不标注内容
2. ❌ 框线不对齐（影响可读性）
3. ❌ 把所有细节都画出来（线框图重在结构，不是高保真）
4. ❌ 在线框图里写交互说明（交互说明应放在文字部分）

## 字符参考表

| 字符 | 用途 |
|------|------|
| `┌ ┐ └ ┘` | 四个角 |
| `├ ┤` | 左右分隔点 |
| `┬ ┴` | 上下分隔点 |
| `─ │` | 横线 / 竖线 |
| `┼` | 十字交叉 |
| `[xxx]` | 按钮 |
| `[xxx ▼]` | 下拉框 |
| `[ ]` 或 `[___]` | 输入框 |
| `☐ ○` | 多选 / 单选 |
| `*` | 必填标识 |
| `+` | 新建 |
| `<` `>` | 返回 / 翻页 |
| `×` | 关闭 |

# FlareFlow App · 需求文档

本仓库**仅**存放产品需求文档（PRD），按批次与需求名组织。  
**所有 AI 生成或落盘的 PRD 只能放在本仓**（https://github.com/king460357/ff-app-prd），不得写入其它仓库。

```
docs/
  {YYYYMMDD}/
    PRD/
      {需求名}/
        *_PRD.md          ← 真理源（主文档）
        *_PRD.html        ← 评审用派生（若有）
        README.md         ← 该需求目录说明（若有）
        assets/           ← 截图等参考（若有）
```

PM 写作工作流与 Skill 见：**[ff-app-prd-skill](https://github.com/king460357/ff-app-prd-skill)**。

1. 产品概述（摘要）

VISITOR 未同意隐私 → 无 UID，SDK 最小化。GUEST 已同意未登录 → guest_uid + Guest-Session-Token，点赞/列表/进度全开。LOGGED_IN → 仅此态可完成 真实货币充值。免费集均可播；付费集均不可播整片（服务端须校验 episode_pay_type）。

完整背景、非目标、§1.6 研发自检与流程图（Mermaid）见 .md 第 1–3 章。

§10 已拍板默认策略（摘要）

  • 10.1 合并：登录成功携带 guest_session_token；冲突以 updated_at 较新者胜；完成后吊销 guest token。
  • 10.2 撤回：仅 GUEST 可见入口 → VISITOR + 吊销；默认软删对终端不可见。
  • 10.3 崩溃：VISITOR 可上报无 user id / 无 IDFA 的 stack + app/os 版本。
  • 10.4 门闸登录后：回剧集上下文，不自动 IAP。

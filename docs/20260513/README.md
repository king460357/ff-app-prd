## 关键决策摘要（执行前必读）

1. **访客模式 ≠ 游客 UID**：访客模式下**不分配任何 UID**；仅在用户**同意隐私条款**且**未登录**时分配**游客 UID**。
2. **访客模式采集**：不收集设备号、广告 ID、行为日志（定义以法务最终口径为准）。
3. **播放**：访客模式与游客 UID 均遵循业务规则——**免费集可看，付费集不可看**（付费集解锁走登录后商业化链路，本 PRD 不展开收银台细节）。
4. **游客 UID 能力**：用户同意条款后，**点赞 / My List（收藏与历史）/ 播放进度同步**等对客功能按本 PRD 全开（依赖条款披露范围）。
5. **充值**：仅**已登录**可完成支付；访客仍可通过 **§3.1**（冷启、三 Tab、待解锁页、顶栏等）完成同意 → `GUEST` → 再登录充值；**不得以「仅登录」替代勾选**。
6. **访客主导航**：`VISITOR` **仅 Discover 驻留**；另三 Tab 与顶栏「退出访客模式」→ **弹窗①**（与冷启同一套）；点 **不同意** → **弹窗②**；**保留搜索**。
7. **实现**：以访客 PRD **§0–§10** 为准；**§3.1** 为「哪些操作出哪个弹窗」主表；**§2.1** 为登录合并等补充约定；与隐私政策冲突走法务变更单。

# requirements-spec · 开屏与免费集插页 AB

## S-01 · 开屏实验

**Key**：`ios_open_ads-test` / `android_open_ads-test`

| 参数 | 行为 |
|------|------|
| **0** | 仅运营开屏，无开屏广告 |
| **1** | 仅开屏广告，禁止运营开屏；单次启动二选一 |

**子开关默认值**（仅参数=1 时读；来源：竞品 `start_ad_switch` + 现网 `app_open`）：

| key | 默认值 |
|-----|--------|
| `hot_start_enable` | **0**（仅冷启动） |
| `inter_time_sec` | **60**（秒） |
| `max_per_session` | **10** |

**预加载 / 上限**（主 PRD 5.1.7）：`app_open` 必预加载；展示后预加载下一发；缓存跟现网/SDK。

**Logo / 拉货**（主 PRD 5.1.6，沿用 FF 1.2.6）：Logo **固定 2 秒**；有缓存 → **满 2 秒后**展示；无缓存 → **0～2 秒**拉货，无货**再延长 5 秒**，**7 秒**仍无货直进首页。

**验收**：见主 PRD **5.1.9**（Gherkin 场景名，无 AC 编号）

## I-01 · 插页实验

**Key**：`ios_free_interstitial-test` / `android_free_interstitial-test`

| 参数 | 行为 |
|------|------|
| **0** | 播放免费剧集时不插页 |
| **1** | 播放免费剧集时按子开关隔集插页 |

**子开关默认值**（仅参数=1 时读；来源：本期 1–10 集隔 2 集 + 竞品免费剧隔集插屏逻辑）：

| key | 默认值 |
|-----|--------|
| `start_episode` | **1** |
| `end_episode` | **10** |
| `interval_episodes` | **2** |
| `inter_time_sec` | **0**（不额外按秒限） |
| `max_per_play_session` | **0**（不另设会话总 cap） |

**预加载 / 上限**（主 PRD 5.2.6）：`immersion_free_int` 必预加载。

**验收**：见主 PRD **5.2.8**（Gherkin 场景名，无 AC 编号）

## User Stories（摘要）

- **US-1**：独立 key 验证开屏顺序。
- **US-2**：参数=1 时冷启动只开屏广告、无运营开屏。
- **US-3**：独立 key 验证免费集插页。
- **US-4**：参数=1 时第 2/4/6… 集可能无奖励插页。

## 广告位

- S-01：`app_open`
- I-01：`immersion_free_int`
